# From Knowledge to Context

This folder is a self-contained Zenodo deposit package for the technical whitepaper **From Knowledge to Context: Building a Google Open Knowledge Format Pipeline That Feeds a Local, Private MCP**.

## Overview

The paper describes how Google Cloud's Open Knowledge Format (OKF) and the Model Context Protocol (MCP) can be combined into a portable, private knowledge harness for AI agents. It covers OKF authoring and publication, MCP delivery patterns, local and private deployment, security, multi-model workflows, and human review.
